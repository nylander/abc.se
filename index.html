Programs/Scripts/Code

Note: All code are now on my GitHub page: https://github.com/nylander

  • Vim Alignment highlighting. Vim highlighting files for DNA sequence alignments. SCREENSHOT. [WIN/MACOSX/UNIX]

  • Vim Nexus syntax highlighting. Nexus syntax highlighting files for the Vim editor. Recognizes many/most of the PAUP/MrBayes commands. SCREENSHOT. [WIN/MACOSX/UNIX]

  • Vim tips for phylo data. A small collection of scripts for working with phylo data in Vim/gVim. vim-phylo.vim. Demo movie (screen capture in FLV format, no sound) on using vim-phylo.
